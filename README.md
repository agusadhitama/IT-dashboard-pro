# IT Dashboard PRO

**IT Dashboard PRO** adalah halaman web dashboard monitoring sederhana yang menampilkan informasi teknis seperti penggunaan CPU, RAM, disk, traffic jaringan, alamat IP, dan jam real‑time.  
Dirancang sebagai **dashboard status sistem statis** untuk visualisasi cepat metrik server/PC secara front‑end oleh *Agus Satria Adhitama*.

---

## 🚀 Live Demo

🌐 https://agusadhitama.github.io/IT-dashboard-pro/

---

## 📌 Deskripsi

IT Dashboard PRO menampilkan panel metrik IT umum seperti:

- **CPU Usage** – Persentase pemakaian CPU real‑time  
- **RAM Usage** – Persentase pemakaian memori  
- **Disk Usage** – Total penggunaan penyimpanan  
- **IP Address** – Deteksi alamat IP klien  
- **Network Traffic & Speed Test** – Indikator traffic dan speed test sederhana  
- **Hacker Terminal** – Panel terminal gaya *hacker aesthetic*  
- **Live Clock** – Jam digital real‑time

Dashboard ini cocok sebagai **prototype tampilan monitoring** untuk dashboard admin IT atau sebagai tema UI belajar.
